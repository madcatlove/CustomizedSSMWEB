그래픽스에서 가장 중요한 개념은 '좌표계 변환' 입니다. 물체의 정점은 각종 좌표변환 행렬이 곱해져서 새로운 좌표값으로 바뀌어 최종적으로 화면에 그려집니다.

따라서 물체를 옮기기 위해서는 전역 좌표계에 변환 행렬을 곱하여 물체의 좌표값을 변환할 수 있습니다.

좌표계를 연속적으로 움직여 원하는 위치와 방향으로 가져다 놓은 다음, 최종적인 좌표계를 기준으로 물체를 그립니다.

Trnaslate 함수를 이용하여 좌표축(원점)을 옮길 수 있습니다.

항등행렬(Identity Matrix)은 원점을(0,0,0)으로 초기화 시킬때 사용합니다.

예시

  1. Traslate(3,0,0);
    //변환 행렬을 먼저 적용시킵니다
    // x+축의 방향으로 3만큼 원점(0,0,0)을 옮깁니다. -> 원점의 좌표(3,0,0)
  2. drawBox(1,0,0,0);
    // 원점(3,0,0)에 box를 그립니다.

중요 !

 지엘에서는 함수를 호출한 순서가 매우 중요합니다. 먼저 호출한 함수가 나중에 호출한 함수들에게 영향을 미치기 때문입니다.

 예를 들면 색을 바꾸기위해선 Color 함수를 호출한 후, drawBox 함수를 호출해 주어야 색깔이 바뀌어서 그려지게 됩니다.

 따라서 Translate로 이동을 시킨 다음에 drawBox로 그려주어야 좌표변환이 적용이 된다는 것에 주의해주세요.

Notice

지엘에서는 translate만 이용하여 3D 오브젝트를 옮길 수 있지만, 튜토리얼에서는 이해를 돕기 위해 x,y,z 값을 추가하여 올길 수 있도록 하였습니다.
