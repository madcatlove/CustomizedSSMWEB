Translate가 정점의 좌표를 옮길 수 있는 반면, Rotate 함수는 정점의 각도를 바꾸는 함수 입니다.

Rotate 함수는 Rotate(angle,x,y,z) 총 4개의 파라미터를 가지고 있는데 angle은 각도, x,y,z는 방향벡터를 나타냅니다. 예를 들어, 값이(45,1,0,0)이면, x축 기준으로 45도 회전시킵니다.

예시

  1. Rotate(45,0,1,0);
    // 좌표축을 Y축을 기준으로 시계 방향으로 45도 회전 시킵니다.
  2. Traslate(4,0,0);
    // x+축의 방향으로 3만큼 원점(0,0,0)을 옮깁니다. -> 원점의 좌표(2,0,2)
  3. drawBox(1,0,0,0);
    // 원점(2,0,2)에 box를 그립니다.

중요 !

Translate와 Rotate의 순서에 주의합니다. Translate를 먼저할 경우 이동한다음 원점의 좌표를 회전시키는 반면, Rotate를 먼저할 경우 회전 시킨후 이동하기 때문에 결과가 달라집니다.
